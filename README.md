# EasyInfo(INS205Assesment2)
A web application that fetches resources for almost anything you want to learn.

HOW TO USE:

1. Type in whatever you would like to learn into the searchbar.

2. Click the search button.

NOTE: THIS WEB APP NEEDS AN INTERNET CONNECTION

The web app will provide you with the top youtube videos, and books on whatever you typed in.

PLEASE USE A FEW SEARCHES AS POSSIBLE. THERE IS A LIMIT TO THE AMOUNT I HAVE FOR VIDEOS

Enjoy :P
